project text

Thoughts

As a first step, we took the original design of the brand as a reference and thought about how we wanted to change, what we wanted to achieve with it and what feeling this brand wanted to convey.
Was decided to create four flavours of this soda (Quatro), using quite striking colours and an original design according to the flavours.

More about the porject

image

For this project, Cinema 4D was used to create the can, Illustrator and Photoshop to design the label box packaging, and the website headers, and After Effects to make the promotional video.

image

project text

The final result is a fun, colorful and healthy brand and good layout website.

project text

Outcome

It was a really exciting and fun project for me, it involved a lot of research and detailed interaction work. Still, I learned some important takeaways from this project related to design and group work.


How to adapt to changing requirements

The different ideas, designs and concepts change. I had to adapt to those changes and still deliver the best possible design on time with tight deadlines.


Always strive for good UX

Creating the wireframe for this project made me understand what is essential to have a good user experience.


Choose what we won't do

There were many excellent ideas or visions that we were able to address with a wide set of functions for the website, animations here and there. However, each of them was unrealistic according to the time. We had to determine where the real value of the website and the brand was so as not to spread ourselves too thin.
